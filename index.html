⚙️ Configuración del sistema

🤖 API Key de Anthropic (IA)

Necesaria para leer facturas en PDF automáticamente. Se guarda solo en este dispositivo (localStorage), nunca en la nube. Obtené tu clave en console.anthropic.com.

🤖 API Key de Google Gemini (Leer facturas PDF)

Se usa para extraer datos de facturas en PDF automáticamente. Gratis hasta 1500 requests/día.
Creá tu clave gratis en aistudio.google.com → "Create API Key".

🗑️ Borrar API Key Anthropic

Elimina la clave de Anthropic guardada de este dispositivo.

📧 EmailJS — Envío de facturas por mail

Permite enviar facturas por mail con link de descarga del PDF. Creá tu cuenta gratis en emailjs.com (200 mails/mes gratis). La plantilla debe tener estas variables:
to_email · proveedor · numero · importe · descripcion · fecha · pdf_url
Incluí <a href="{{pdf_url}}">Descargar PDF</a> en el cuerpo del mail.

📥 Lectura automática de Gmail

Permite leer las facturas que te llegan por mail. Pegá el ID de cliente de OAuth de Google Cloud (termina en .apps.googleusercontent.com). Lo sacás en Google Cloud → Credenciales → "ID de cliente de OAuth 2.0" → Web client.
